⚙️ AI 연결 설정 (선택)

내부/외부 LLM API를 연결하면 더 정교한 피드백을 받을 수 있습니다. 연결하지 않아도 키워드 기반 로컬 체크로 학습이 가능합니다.

🔒 공개 GitHub Pages에서는 API Key가 브라우저(localStorage)에 저장되므로 민감한 키를 입력하지 마세요. 개인 학습용/로컬 테스트용으로만 사용하세요.

➕ 스크립트 카드 추가/관리

나만의 OPIc 스크립트를 추가하세요. 추가한 카드는 이 브라우저에 저장되어 '내 카드' 카테고리에 모입니다.

💡 키워드를 입력하면 비교 정확도가 높아집니다. 비워두면 스크립트에서 자동 추출합니다.

JSON 파일로 여러 카드를 한 번에 가져옵니다. 아래 형식을 따르세요.

[

    "cat": "exp",

    "hint": "경험을 시간순으로 묘사",
    "keyEnglish": "It was the most memorable trip.",
    "fullScript": "Last spring, I visited Jeju...",
    "keywords": ["Jeju", "sunrise", "peaceful", "memorable"]
